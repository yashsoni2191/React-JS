(7) - What is component?

Ans - Components are independent and reusable bits of code. They serve the same purpose as JavaScript functions, but work in isolation and return HTML. Components come in two types, Class components and Function components.

(8) - What is Hooks?

Ans - Hooks allow function components to have access to state and other React features. Because of this, class components are generally no longer needed. Hooks are a function which is used with functional components to handle and manage the states and side-effects. So state is mutable or changeable.
